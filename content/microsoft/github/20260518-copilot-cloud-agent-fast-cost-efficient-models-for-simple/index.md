When you delegate a task to Copilot cloud agent, you can pick the model it uses to do its work.
Today, we&rsquo;re expanding the list of supported models to include faster, more cost-efficient options. This means you can pick the right model for the job: a smaller, quicker model for straightforward changes, or a more capable model for complex work.
The following models are now available, in addition to the existing options:

Claude Haiku 4.5 (0.33x multiplier)
GPT-5.4-mini (0.33x multiplier)

To learn more, see &ldquo;Changing the AI model for Copilot cloud agent&rdquo; in the GitHub Docs.
